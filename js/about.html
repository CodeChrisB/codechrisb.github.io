How to create thousands accounts on one service?

            I think this is obvious I used VPN, just for saftey. But this still does not anwser how I was able
            to create that many accounts.

            I am not sure how I found out about it but gmail lets you create 
            2^(n-1) Accounts were n stands for the characters in your email before the @gmail.colum

            To explain this: 
            Let's suppose we have the email adress max@gmail.com
            I can create with this email in 2^(3-1) = 2^2 = 4 Accounts on most services by simply
            adding points between the characters. Like this :

            max@gmail.com
            m.ax@gmail.com 
            m.a.x@gmail.com 
            ma.x@gmail.com
